Remarks

To override the default method used to image decoded spans please set FAXFILLFUNC tag with an instance of this delegate.

Fill methods can assume the runs array has room for at least width runs and can overwrite data in the runs array as needed (e.g. to append zero runs to bring the count up to a nice multiple).
